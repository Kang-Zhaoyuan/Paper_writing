# Paper Written by Our Team for the 2026 Mathematical Contest in Modeling (MCM), problem C
# 2026年美国大学生数学建模竞赛C题论文
This repository contains the LaTeX source and plotting scripts for the paper in this folder.

本仓库包含本文件夹中论文对应的 LaTeX 源文件与绘图脚本。

It includes figures generated from the provided CSV datasets under the current directory and `images/`.

其中包含由当前目录及 `images/` 下提供的 CSV 数据集生成的图表。

Main documents: `paper.tex` and `paper_English.tex`.

主要文档为：`paper.tex` 与 `paper_English.tex`。

Run the Python scripts to reproduce plots, then compile the LaTeX files to build the PDFs.

运行 Python 脚本可复现绘图结果，随后编译 LaTeX 文件即可生成 PDF。
